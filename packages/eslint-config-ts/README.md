# @fastcoder/eslint-config-ts



Using npm:

```shell
$ npm install -D @fastcoder/eslint-config-ts
```

Using yarn:

```shell
$ yarn add -D @fastcoder/eslint-config-ts
```

Using pnpm:

```shell
$ pnpm add -D @fastcoder/eslint-config-ts
```

Just introduce in  .eslintrc

```js
  "extends": [
    "@fastcoder/eslint-config-ts"
  ]
```
