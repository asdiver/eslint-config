# @fastcoder/eslint-config-base



Using npm:

```shell
$ npm install -D @fastcoder/eslint-config-base
```

Using yarn:

```shell
$ yarn add -D @fastcoder/eslint-config-base
```

Using pnpm:

```shell
$ pnpm add -D @fastcoder/eslint-config-base
```

Just introduce in  .eslintrc

```js
  "extends": [
    "@fastcoder/eslint-config-base"
  ]
```
