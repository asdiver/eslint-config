# @fastcoder/eslint-config



Using npm:

```shell
$ npm install -D @fastcoder/eslint-config
```

Using yarn:

```shell
$ yarn add -D @fastcoder/eslint-config
```

Using pnpm:

```shell
$ pnpm add -D @fastcoder/eslint-config
```

Just introduce in  .eslintrc

```js
  "extends": [
    "@fastcoder/eslint-config"
  ]
```
