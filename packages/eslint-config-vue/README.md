# @fastcoder/eslint-config-vue



Using npm:

```shell
$ npm install -D @fastcoder/eslint-config-vue
```

Using yarn:

```shell
$ yarn add -D @fastcoder/eslint-config-vue
```

Using pnpm:

```shell
$ pnpm add -D @fastcoder/eslint-config-vue
```

Just introduce in  .eslintrc

```js
  "extends": [
    "@fastcoder/eslint-config-vue"
  ]
```
