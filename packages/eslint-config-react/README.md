# @fastcoder/eslint-config-react



Using npm:

```shell
$ npm install -D @fastcoder/eslint-config-react
```

Using yarn:

```shell
$ yarn add -D @fastcoder/eslint-config-react
```

Using pnpm:

```shell
$ pnpm add -D @fastcoder/eslint-config-react
```

Just introduce in  .eslintrc

```js
  "extends": [
    "@fastcoder/eslint-config-react"
  ]
```
